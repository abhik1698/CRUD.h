Create Table Syntax

MySQL syntax:

CREATE TABLE tableName( id INT, name VARCHAR(40) );

MySQL Shorthand syntax:

c tableName id name

Drop Table Syntax

MySQL:

DROP TABLE tableName;

MySQL Shorthand:

dr tableName

Drop Schema Syntax

MySQL:

DROP DATABASE databaseName;

MySQL Shorthand:

r

Insert Row Syntax

MySQL:

INSERT INTO tableName VALUES( 1, 'name' );

MySQL Shorthand:

i tableName 1 name

Update Row Syntax

MySQL:

UPDATE tableName SET column = value where column1 = value1;

MySQL Shorthand:

u tableName column1 value1 column value

Delete Row Syntax

MySQL:

DELETE FROM tableName WHERE id = 1;

MySQL Shorthand:

d tableName id 1

Truncate Table Syntax

MySQL:

TRUNCATE TABLE tableName;

MySQL Shorthand:

t tableName

SQL

SQL is a standard language for accessing and manipulating databases.

What is SQL?

  • SQL stands for Structured Query Language
  • SQL lets you access and manipulate databases

    • What Can SQL do?

  • SQL can execute queries against a database
  • SQL can create & delete a database.
  • SQL can insert, read, update, delete, records in a database
